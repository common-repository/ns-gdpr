== Description ==

**ACTIVATION THIS PLUGIN DOES NOT GUARANTEE YOU FULLY COMPLY WITH GDPR. PLEASE CONTACT A GDPR CONSULTANT OR LAW FIRM TO ASSESS NECESSARY MEASURES.**

You have to configure this plugin to help you provide GDPR compliance.
Page option will appear in your WordPress backend menu, and here you can choose to enable GDPR checkbox for WordPress comments, WooCommerce and also Contact from 7!
You can choose text to display for the checkbox and an "error text" if users don't accept GDPR checkbox.


*We test our plugins with many other. If you find incompatibility with one specific plugin, please inform us at <a href="mailto:report@nsthemes.com">report@nsthemes.com</a>.*


== Installation ==

1. Upload the entire 'ns-gdpr' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'NS GDPR' and enable GDPR checkbox


== Frequently Asked Questions ==

= Code is required to use this plugin? =

No code require to use this plugin.

= Is this plugin compatible with WooCommerce? =

Yes, you can view GDPR checkbox in WooCoommerce checkout after enable it.

= Can I use it only with WordPress comments? =

Yes, you can.

= Is this plugin compatible with Contact form 7? =

Yes, you can view GDPR checkbox in CF7 forms after enable it.
